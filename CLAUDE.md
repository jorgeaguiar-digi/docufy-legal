# Docufy Legal

## Propósito

Proyecto independiente para Docufy Legal — software de gestión para despachos de abogados especializados en accidentes de tráfico.

Este repositorio centraliza toda la información del producto: descripción, funcionalidades, público objetivo, propuesta de valor, estrategia comercial y cualquier material generado (presentaciones, propuestas, documentación).

---

## Estructura

```
docufy-legal/
├── CLAUDE.md               ← este archivo (contexto del proyecto)
├── knowledge/              ← base de conocimiento del producto
│   ├── product.md          ← descripción completa del producto
│   ├── market.md           ← mercado, ICP, competencia
│   └── pricing.md          ← modelo de precios
├── presentations/          ← decks y presentaciones generadas
├── proposals/              ← propuestas comerciales
└── docs/                   ← documentación técnica y funcional
```
